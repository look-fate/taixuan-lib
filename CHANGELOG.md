# Changelog

## [1.1.0] - 2026-03-17

### Added
- 新增 `generateMultiple()` 函数，支持批量起卦
- 新增 `searchByName()` 函数，支持按名称搜索卦象
- 新增 `formatResult()` 函数，格式化占卜结果输出
- 新增 `getAllGua()` 函数，获取所有卦象数据
- 新增 `GUA_MAP` 常量，提供 O(1) 查询性能

### Changed
- 优化 `lookupByIndex()` 查询性能，从 O(n) 提升到 O(1)
- 重构起卦方法，提取公共逻辑减少代码重复
- 增强 `parseCode()` 类型安全，添加边界检查
- 增强 `lookupByIndex()` 参数验证，拒绝无效索引

### Fixed
- 修复 `parseCode()` 可能产生无效编码的问题
- 修复短编码处理，使用 `padStart` 补齐位数

### Tests
- 新增边界值测试（负数、小数、无效索引）
- 新增确定性测试（数字法）
- 新增概率分布测试（骰子法）
- 新增实用功能测试
- 测试覆盖率从 5 个用例提升到 14 个

### Performance
- 查询性能提升最多 81 倍（最坏情况）
- 代码重复减少约 40%

## [1.0.0] - 2026-03-17

### Added
- 初始版本发布
- 5 种起卦方法（随机、蓍草、骰子、硬币、报数）
- 完整 81 首卦象数据（含赞辞、爻辞）
- TypeScript 类型支持
- ESM 和 CommonJS 双格式支持
